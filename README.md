# KDA_Pikachu

## Team package files

This repository contains the team package for the Python collaboration task.

### Leader role

- `__init__.py`: exposes teammate functions as a package.
- `__main__.py`: runs sample calls for each teammate function.

### Member roles completed here

- `multiplication.py`: member 3 multiplication function.
- `division.py`: member 4 division function.

Run the package with:

```bash
python __main__.py
```
